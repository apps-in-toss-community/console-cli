# console-cli

> 🚧 **Work in Progress** — not yet published.
> 아직 개발 중입니다. 릴리스 전입니다.

CLI and MCP server for the [Apps in Toss](https://toss.im/) console.

앱인토스 콘솔을 **CLI**와 **MCP**로 자동화하는 커뮤니티 도구.

> This is an **unofficial, community-maintained** project. Not affiliated with or endorsed by Toss or the Apps in Toss team.
> 이 프로젝트는 **비공식 커뮤니티 프로젝트**입니다. 토스/앱인토스 팀과 제휴 관계가 아닙니다.

## Goal / 목표

최초 로그인만 브라우저로 하고, 이후에는 **headless 브라우저로 빌드·배포·릴리스**를 커맨드 한 줄 또는 Claude/Codex 명령으로 처리할 수 있게 한다.

Log in once in a browser, then drive builds, deploys, and releases from your shell or from an AI coding agent via headless automation.

`agent-plugin`의 `/ait deploy` 등이 내부적으로 이 CLI를 호출하는 구조를 목표로 한다.
